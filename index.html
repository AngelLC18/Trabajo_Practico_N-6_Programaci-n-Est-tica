3) Manejo de variables:

Las variables se pueden declarar usando var, const, o let. var tiene alcance de función, mientras que const y let tienen alcance de bloque. Las variables const no pueden ser reasignadas, mientras que las variables var y let si pueden serlo.

Tipo de datos

  • Primitivos: Son los valores que no pueden ser cambiados:

    • 1) Numerico: Almacena valores enteros o decimales. ej var = edad = 19;
    • 2) String: Almacena valores tipo texto. ej= var nombre = Angel;
    • 3) Boolean: Almacena valores verdaderos, falsos y compraciones. ej: var boleano = true;
    • 4) Symbol: Se utilizan por lo general para añadir claves de propiedades únicas a un objeto que no sean iguales a las claves que cualquier otro código pueda añadir al objeto. ej let sym = Symbol()
    • 5) Undefined: Almacena un valor que todavia no esta definido, utilizando la n al final de un número entero. Ej: var nombre;
    • 6) BigInt: Representa números enteros con precisión arbitraria. Ej: const num = 2n ** 53n; 9007199254740992n;
  • 7) Object: new object, new array, new map, new set, new weakmap, new weak set, new date: Almacena un conjunto de elementos de cualquier tipo. ej= var persona ["Angel " , "19 "];
  • 8) Null: Permite borrar el valor de una variable. ej var nombre = angel; var nombre = null;
  • 9) NaN: Muestra valores invalidos debido a un error. ej multiplicar el valor de una cadena de texto.
  • 10) Function: Fragmento de código que puede ser llamado por otro código o por sí mismo, o por una variable que haga referencia a la función. ej: function funcion() {};, const funcion ();
