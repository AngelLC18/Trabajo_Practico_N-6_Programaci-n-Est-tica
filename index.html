3) Manejo de variables:

Las variables se pueden declarar usando var, const, o let. var tiene alcance de función, mientras que const y let tienen alcance de bloque. Las variables const no pueden ser reasignadas, mientras que las variables var y let si pueden serlo.

Tipo de datos

  • Primitivos: Son los valores que no pueden ser cambiados:

    • 1) Numerico: Almacena valores enteros o decimales. ej var = edad = 19;
    • 2) String: Almacena valores tipo texto. ej= var nombre = Angel;
    • 3) Boolean: Almacena valores verdaderos, falsos y compraciones. ej: var boleano = true;
    • 4) Symbol: Se utilizan por lo general para añadir claves de propiedades únicas a un objeto que no sean iguales a las claves que cualquier otro código pueda añadir al objeto. ej let sym = Symbol()
    • 5) Undefined: Almacena un valor que todavia no esta definido, utilizando la n al final de un número entero. Ej: var nombre;
    • 6) BigInt: Representa números enteros con precisión arbitraria. Ej: const num = 2n ** 53n; 9007199254740992n;
  • 7) Object: new object, new array, new map, new set, new weakmap, new weak set, new date: Almacena un conjunto de elementos de cualquier tipo. ej= var persona ["Angel " , "19 "];
  • 8) Null: Permite borrar el valor de una variable. ej var nombre = angel; var nombre = null;
  • 9) NaN: Muestra valores invalidos debido a un error. ej multiplicar el valor de una cadena de texto.
  • 10) Function: Fragmento de código que puede ser llamado por otro código o por sí mismo, o por una variable que haga referencia a la función. ej: function funcion() {};, const funcion ();

Expresiones Regulares:

Son patrones que se utilizan para hacer coincidir combinaciones strings. las expresiones regulares también son objetos. se utilizan con los métodos exec() y test() de RegExp, y con match(), matchAll(), replace(), replaceAll() (en-US), search()
y split() métodos de String.

4) Las forma de declarar una funcion en js son las siguientes:

  • Function declaration: Este tipo de función se creará con la palabra reservada function, seguido obligatoriamente por un nombre, que identificará a nuestra función, una lista de parámetros entre paréntesis, y el símbolo de las llaves {}. Qué será el que delimite el contenido de nuestro conjunto de sentencias.

  • Function expression: La sintaxis para crear estas funciones es similar a la anterior. La única diferencia es que la definición de nuestra nueva función no comienza por instrucción function y el nombre de la función es opcional.

  • IIFE: Para repetir la función en una sola ocasión y obtener un resultado.se puede utilizar la funciones IIFE. Este tipo de funciones se ejecutan inmediatamente y no son accesibles posteriormente.

  • Shorthand method definition: Esta forma de crear las funciones puede ser utilizado como método en la declaración de un objeto o en las clases de ES6. Para crear este tipo de función debemos asignar un nombre de función seguido de una lista de parámetros entre paréntesis y los símbolos de llaves para delimitar el cuerpo de las instrucciones.

  • Arrow function: La forma de crear estas funciones es la siguiente: Primero definiremos la lista de parámetros, en caso de ser necesario, entre paréntesis seguido del símbolo => y las {} para indicar las instrucciones que se van a realizar.

  • Generator function: Si queremos salir con antelación de una función debemos ejecutar un return, o lanzar un error. Por lo que las instrucciones que se encuentren a continuación del return o el error no se ejecutarán. Esta funcion nos permite parar/salir de la función en un punto dentro del conjunto de instrucciones que forman nuestra función y retornar posteriormente la ejecución desde el punto en el que paramos anteriormente. La sintaxis de estas funciones es igual a las "function declaration " o "function expression ". utilizando el símbolo * al comienzo de la definición de nuestra función.

  • Function constructor: La función constructora es técnicamente una función normal. Aunque hay dos convenciones: Son nombradas con la primera letra mayúscula. Sólo deben ejecutarse con el operador "new ".
    Cuando una función es ejecutada con new, realiza los siguientes pasos: Se crea un nuevo objeto vacío y se asigna a this. Se ejecuta el cuerpo de la función. Normalmente se modifica this y se le agrega nuevas propiedades. Se devuelve el valor de this.

5) Conversiones entre tipos:

  • ToString: Cuando se necesita representar en forma de texto un valor, se puede ralizar con String(valor).

  • To Number: Ocurre automaticamente en funciones matematicas e expresiones, como por ejemplo cuando se dividen valores no numéricos, se puede realizar con Number(valor).

  • ToBoolean: Ocurre en operadores lógicos, o llamando a la función, se puede realizar con Boolean(value). La conversión es la siguiende : Valores que son 0, " ", null, undefined y NaN, se convierten en false y otros valores se convierten en true, aunque el string con un 0 es true en JavaScript

Operadores

  • Ariméticos: Toman valores numéricos como sus operandos y retornan un valor numérico.
    • +: Suma
    • -: Resta
    • /: División
    • *: Multiplicación
    • %: Residuo
    • **: Exponenciación
  • Unarios: Operación con un solo operando.
    • delete: Elimina una propiedad de un objeto.
    • void: Descarta el valor de retorno de una expresión.
    • typeof: Determina el tipo de un objeto dado.
    • +: Convierte su operando al tipo Number.
    • -: Convierte su operando al tipo Number y luego lo niega.
    • ~ : No bit a bit.
    • ! : No lógico.
  • Relacionales: Compara sus operandos y devuelve un valor booleano si la compración es verdadera o no.
    • in: Determina si un objeto tiene una determinada propiedad.
    • instanceof: Determina si un objeto es una instancia de otro objeto.
    • < : Menor que.
    • > : Mayor que.
    • <=: Menor o igual a.
    • >= : Mayor o igual a.
  • De desplazamiento de bits: Cambia todos los bits del operando.
    • << : Bit a bit a la izquierda.
    • >> : Bit a bit a la derecha.
    • >>> : Bit a bit a la derecha sin signo.
  • de igualdad: Un operador de igualdad siempre es booleano si la compración es verdadera.
    • == :Igualdad.
    • != : Desigualdad.

    • !== : Desigualdad estricta.
  • Binarios bit a bit: Trtan a sus operandos como conjuntos de 32 bits y retornan valores numéricos estandar de js.

    • | : "o " bit a bit.

  • Lógicos binarios: Utilizan generalmente valores booleanos, cuando es asi, devuelven un valor booleano.

  • Condicional: Retorna uno de dos valores, según el valor lógico de la condición.

  • De Asignación: Asigna un valor a su operando izquierdo, tomando como referencia el valor de su operando derecho.

6) Las estructuras de control son:

  • If: Si ocurre algo, haz lo siguiente

  • If/Else:Si ocurre algo, haz esto, sino, haz lo esto otro

  • Switch: Se utiliza para hacer diferentes acciones basadas en diferentes condiciones.

  • While: Mientras se cumpla la condición realizar estas acciones. Cuando la condición deje de cumplirse salimos del bucle.

  • Do While: Ejecute el bucle hasta que se cumpla la condición.

  • For: Se utiliza para repetir una o más instrucciones un determinado número de veces

  • Try Catch: Try: Intenta un bloque de intrucciones y especifica una respuesta, en caso de una excepción se ejecuta lo que este dentro del bloque catch.

  • Break y Continue: Break: Salir/Saltar fuera un bucle, finalizándolo, Continue: Saltar hasta la siguiente iteración del bucle. Permite saltar una o más iteraciones.
